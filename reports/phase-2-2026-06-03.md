**Merge commit:** c37f7a5

- Open-canvas Sonar fetch — the 18-domain `search_domain_filter` allowlist is gone; one `news:` query per volume-safe/borderline **section** (6 of them), each article attributed to that section.
- `services/content_classifier.py` — one cheap Haiku call scores each article `is_news` (drops vendor promo) + `importance` 0–2 (operator-relevance). Fails open on API error (treat as news, importance 1) so a hiccup never kills a run.
