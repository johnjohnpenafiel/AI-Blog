# Post · Section 4 — Related Dispatches

Up to three other dispatches as "Fig." cards (same section first, then most recent). Hidden entirely when there are no other posts.

- **Source:** `frontend/src/components/public/related-dispatches.tsx`
- **Gutter marker:** `(rel)`

```
Related Dispatches band
├── Rule Header                ← pulse dot · "RELATED DISPATCHES" · hairline · "NN READS"

        ├── Fig Number         ← "FIG. 1/2/3" tag on the image slot
        └── Section Line       ← ◆ SECTION shown (showSection variant)
```
